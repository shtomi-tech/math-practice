# 数学ミニ試験 問題対応計画

## 目的

各オリジナル問題が、参考資料のどの問題で扱われる技能に対応するかを明示する。
参考資料の問題文・図・数値を転載せず、対応する技能と独自化の方針だけを管理する。

既存の第1回・第2回は、完成済み問題と資料を照合して対応を復元したものとする。
第3回以降は、下表の対応を生成前に確定し、その後に問題を作成する。

## 生成・採用ルール

1. 各問題には主対応を1件設定する。補助的に使う資料問題がある場合だけ副対応を追加する。
2. 資料から利用するのは技能・解法骨格・典型的な誤りであり、問題文・図・固有の設定は転載しない。
3. 少なくとも「設定」「式構造」「問い方」「条件」のうち2項目を変更する。数値だけの変更は不合格とする。
4. 対応表の割当後に問題を生成し、独立検算と8項目の品質チェックを最大3周行う。
5. 同じ資料問題を同じ問い方で複数回に使わない。
6. 第2回 `m02-4-3` は、第3回追加前に資料と異なる設定へ差し替えて再検算する。
7. PDF本体は `data/` に置き、git・GitHub Pagesへ含めない。

## 1回分の完了条件

- 12問すべてに資料対応と独自化方針がある。
- 4単元・12問・100点・30分の構成を満たす。
- 各問題が独立検算と品質基準8項目をすべて通過する。
- `node --check static/app.js` と `node scripts/check-exam.js` が成功する。
- 公開前に、資料PDF・抽出画像・教材本文がgit対象になっていないことを確認する。
