# 数学ミニ試験

全6回構成を予定し、各回に4単元（式の展開・因数分解、実数・1次不等式、集合と論理、場合の数）を混合する、30分のオリジナル演習アプリです。現在は第1回・第2回を公開しています。

画面上部の「回を変える」から第1回・第2回をページ内で切り替えられます。最後に選んだ回は端末に保存され、次回も同じ回を表示します。URLパラメータにも対応し、`?exam=mini_01` で第1回、`?exam=mini_02` で第2回を直接表示します。受験状態（localStorage）とクラウド保存先は回ごとに分離されます。

## 方針

- `static/data.js` の問題は、参考PDFの問題文を転載せず、問題タイプだけを参考にして新規作成します。
- 試験中は正誤・解説を表示せず、提出後に採点と解説を表示します。
- 数字回答はキーボード入力に加え、クリック式テンキーでも入力できます。
- 通常URLでは受験状態をブラウザの `localStorage` に保存し、生徒別共有URLでは受験中データと採点結果を共通Supabaseへ同期します。
- 画面とA4印刷の両方に対応します。

## 実行

静的ファイルなので、リポジトリ直下で次のように起動できます。

```text
py -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開きます。数式表示はKaTeX CDNを利用しているため、初回表示時はインターネット接続が必要です。

## 問題の追加

`static/data.js` の `groups` に問題を追加します。`numeric`、`choice`、`multi` の3形式に対応しています。各回の合計点が100点、想定時間が30分になるよう調整してください。
